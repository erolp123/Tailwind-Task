Project Summary

This project uses HTML, Tawildcss and REACT framework to bring forward a clean looking, functional and responsive web app for movie lovers where they can search for their favourite movie via the search function with real-time updates

More about the project

In this project I wanted to demonstrate a combination of my UX Design and Front-end development skills. I have achieved this by using Tailwindcss to create a responsive and consistent layout. Tailwindcss has also allowed a faster development compared to regular css all while bringing much consistent code.

React Javascript library has also been used to provide the functionality of the website. In this example the search area and the display of movie cards are being presented via React Javascript. The movie card area is always updated based on trending movies and based on what the user enters to the search field the displayed movies are also updated in real-time.
