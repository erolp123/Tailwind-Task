Movie and Wallet Mobile App Design



Project Summary

This project is a new and updated design of a potential mobile crypto wallet & movie streaming app

More about the project

This project was made to demonstrate my skills in wireframing and prototyping to come up with new and improved design layouts for mobile apps. My design includes thinking about the user side of things and making sure that the users have the best possible experience while using the product. This is done by following up to date design rules and following industry standards.
